### Proyectos destacados

* **[Nance E-Commerce System]**
  Solución *full-stack* completa: el *backend* está desarrollado con **Java Spring Boot** e incorpora autenticación robusta mediante JWT y Spring Security. El *frontend* es una aplicación de página única (SPA) altamente adaptable, construida con **React, TypeScript y Tailwind CSS**, que incorpora gestión de estado para los flujos del carrito y del proceso de compra.

* **[PawSchedule]**
  Aplicación nativa para Android, desarrollada íntegramente en **Kotlin**, con una interfaz de usuario moderna basada en **Jetpack Compose**. Implementa *Clean Architecture* y consume servicios REST externos mediante Retrofit para una gestión fluida de las citas de las mascotas.

* **[Sistema Distribuido de Gestión de Mascotas y Clínicas Veterinarias]**
  Arquitectura robusta de microservicios desarrollada con **Java 21, Spring Boot 3.2.5 y Spring Cloud 2023.0.1**. Implementa patrones arquitectónicos como API Gateway, Config Server, Eureka Service Discovery y Circuit Breaker (Resilience4j) para garantizar la tolerancia a fallos. Todo el proyecto aplica estrictamente el patrón Controller-Service-Repository (CSR) y los principios de *Clean Code*.
